How to use

  1. Clone the repository locally and install dependencies: npm install.
  2. Start the bot: npm start.
  3. Open the local interface at http://localhost:3000 or use the buttons above.
  4. Scan the WhatsApp QR code when prompted to authenticate the bot.

The GitHub Pages page is a static control panel, but the actual bot runs on your local machine.
